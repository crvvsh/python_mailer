Installation Guide

Step 1: Clone Repository

git clone https://github.com/crvvsh/python_mailer.git
cd python_mailer

Step 2: Create Virtual Environment

python3 -m venv venv
source venv/bin/activate # macOS/Linux
venv\Scripts\activate # Windows

Step 3: Install Dependencies

pip install -r requirements.txt

Step 4: Configure Environment

cp .env.example .env

Edit .env file with your SMTP settings (Gmail, Outlook, etc.)

Step 5: Run Application

python web.py

Open browser at http://localhost:5000

⚠️ SMTP Configuration: You need to configure SMTP settings in .env file before sending emails.

How to Use Python Mailer

📧 Adding Subscribers

Via Web Interface:
  1. Click "Subscribers" tab
  2. Click "Add New Subscriber"
  3. Enter email and name
  4. Click "Save"

📝 Composing Email

Via Web Interface:
  1. Click "Compose" tab
  2. Enter recipient email
  3. Add subject and message
  4. Click "Send" or "Schedule"

💻 Using Python API

Add Subscriber Programmatically:

from mailer.subscribers import SubscriberManager

manager = SubscriberManager()
manager.add("user@example.com", "John Doe")

Validate Email:

from mailer.validators import EmailValidator

validator = EmailValidator()
is_valid = validator.validate("user@example.com")
print(is_valid) # True

Send Email:

from mailer.email_sender import EmailSender

sender = EmailSender(...)
result = sender.send(
  to="user@example.com",
  subject="Hello\!",
  body="This is a test email."
)
print(result.success)

Frequently Asked Questions

❓ Jak skonfigurować Gmail SMTP?

1. Włącz 2FA na swoim koncie Google

2. Wejdź na Google App Passwords

3. Wygeneruj hasło dla "Mail" i "Windows Computer"

4. Użyj tego hasła w pliku .env

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=your-app-password

❓ Czy mogę używać Outlook?

Tak\! Użyj następujących ustawień:

SMTP_HOST=smtp-mail.outlook.com
SMTP_PORT=587
SMTP_USERNAME=your-email@outlook.com
SMTP_PASSWORD=your-password

❓ Ile subskrybentów mogę zarządzać?

Python Mailer został testowany z setkami tysięcy subskrybentów. Limit zależy głównie od Twojego SMTP providera.

❓ Czy jest API?

Tak\! Możesz używać API Pythona bezpośrednio lub HTTP API (planowane w v2.0)
